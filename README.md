## Installation

```powershell
scoop bucket add klaas https://github.com/klaas-sh/scoop-bucket
scoop install klaas
```

## Update

```powershell
scoop update klaas
```

## Uninstall

```powershell
scoop uninstall klaas
scoop bucket rm klaas
```

## About

This bucket contains the Scoop manifest for [klaas](https://klaas.sh), a remote access wrapper for Claude Code that enables you to control your sessions from anywhere via a web interface.
